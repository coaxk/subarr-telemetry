What's not on this page

File paths, titles, IPs, hostnames, API keys, languages, anything user-fingerprintable. These cannot reach this page because they cannot reach the database — the receiving worker rejects payloads containing any of them with HTTP 400 (forbidden field: ...), and the database schema has no columns for them.

The exact JSON payload your install sends is always visible in Settings → Telemetry inside subarr. One-click opt-out in the same panel and in the onboarding wizard.
